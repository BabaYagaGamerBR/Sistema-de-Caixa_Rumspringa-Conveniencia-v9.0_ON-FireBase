# Rumspringa Convenience Cash Register System — V9.6.1 Online Firebase Beta

Hybrid POS / Cash Register system developed using pure HTML, CSS, and JavaScript, featuring LocalStorage + manual synchronization through Firebase Firestore REST.

This project was designed with focus on:

* simplicity;
* performance;
* offline functionality;
* multi-device support;
* real-world convenience store usage;
* operation without requiring a dedicated server.
