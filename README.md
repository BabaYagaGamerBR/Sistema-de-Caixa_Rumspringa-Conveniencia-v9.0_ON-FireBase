# Sistema de Caixa Rumspringa Conveniência — V9.6.1 Online Firebase Beta

Sistema de Caixa / PDV híbrido desenvolvido em HTML, CSS e JavaScript puro, com funcionamento LocalStorage + sincronização manual via Firebase Firestore REST.

Projeto criado com foco em:

* simplicidade;
* performance;
* funcionamento offline;
* multi-dispositivo;
* uso real em conveniência/comércio local;
* operação sem necessidade de servidor próprio.

# ☁️ Sistema Online Híbrido (Firebase Beta)

A versão V9.6.4 possui sincronização híbrida via Firebase Firestore REST.

### Funcionamento:

* os dados continuam locais;
* sincronização é manual;
* envio e download via Firestore;
* funcionamento multi-dispositivo;
* mesclagem inteligente de vendas;
* sincronização administrativa centralizada.

### Recursos:

* sincronização segura;
* merge de vendas;
* proteção contra perda de dados;
* sincronização de usuários;
* compatível com GitHub Pages;
* sem servidor dedicado.
